This code creates and populates a database for standard applications, then integrates it with a php/html website.
This website is populated from the database and inserts new application information into the database.

To make this site work:
First, run ProjectTableScript.txt on the database.
Start AMPPS. Make sure PHP and MYSQL are running. Put this in the www folder of 
AMPS. Fill in login credentials in mysql_start.php.
Start with login.php. 
Submit/New Account takes you to the next page.
